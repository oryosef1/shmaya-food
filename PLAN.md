## עיצוב ומראה

### צבעים
- גוון חם/טבעי שמתאים לאווירת האוכל הביתי - שמנת, ירוק זית, כתום חם

### פונטים
- פונט עברי נקי מ-Google Fonts (למשל Heebo או Assistant)

### לייאאוט
- RTL מלא
- מובייל-פירסט (רספונסיבי)

- ניווט קבוע (sticky) למעלה

### גלריות תמונות
- כל גיג עם תמונות מקבל גלריה אופקית נגללת (CSS scroll-snap)
- תמונות בגובה אחיד (~300px מובייל, ~400px דסקטופ)
- אפשרות לחיצה להגדלה (lightbox פשוט ב-JS)

## טכנולוגיה

### קבצים
```
moms-website/
├── index.html          ← קובץ יחיד עם HTML + CSS + JS מוטמע
├── content.txt         ← טקסט מקורי (לרפרנס)
├── PLAN.md             ← קובץ זה
└── images/
    ├── business/       ← לוגו/תמונת עסק (1 תמונה)
    ├── tali/           ← תמונות טאלי (5 תמונות)
    └── suflaki/        ← תמונות סופלקי (5 תמונות)
```

### גישה
- **קובץ HTML יחיד** - CSS ו-JS מוטמעים בתוך הקובץ
- אין צורך בפריימוורקים או build tools
- אפשר לפתוח ישירות בדפדפן (file://) או להעלות לכל שרת סטטי
- תמונות נטענות מהתיקיות המקומיות (נתיבים יחסיים)

### תכונות טכניות
- Semantic HTML5 (`<section>`, `<nav>`, `<header>`, `<footer>`)
- CSS Grid / Flexbox ללייאאוט
- CSS scroll-snap לגלריות
- Smooth scrolling לניווט
- Lightbox פשוט ב-vanilla JS (בלי ספריות חיצוניות)
- Lazy loading לתמונות (`loading="lazy"`)
- Meta tags לשיתוף ברשתות (Open Graph)
